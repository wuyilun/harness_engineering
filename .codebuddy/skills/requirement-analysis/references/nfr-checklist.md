# 非功能需求检查清单

## 性能
- [ ] 页面首次加载时间 ≤ 3s
- [ ] API 响应时间 ≤ 500ms（P95）
- [ ] 支持 xx 并发用户
- [ ] 大数据量列表支持虚拟滚动/分页
- [ ] 图片/资源懒加载

## 安全
- [ ] 用户认证机制（JWT/Session/OAuth2）
- [ ] 角色权限控制（RBAC）
- [ ] 敏感数据加密存储
- [ ] HTTPS 传输
- [ ] CSRF 防护
- [ ] XSS 防护
- [ ] SQL 注入防护
- [ ] 速率限制

## 可用性
- [ ] 浏览器兼容性（Chrome/Firefox/Safari/Edge 最新2个版本）
- [ ] 响应式设计（移动端/平板/桌面）
- [ ] 无障碍访问（WCAG AA）
- [ ] 键盘导航支持
- [ ] 国际化/本地化（按需）

## 可靠性
- [ ] 错误处理完备（无未捕获异常）
- [ ] 友好的错误提示
- [ ] 数据备份策略
- [ ] 故障恢复机制
- [ ] 优雅降级

## 可维护性
- [ ] 代码覆盖率 ≥ 80%
- [ ] API 文档自动生成
- [ ] 日志记录完备
- [ ] 监控告警
- [ ] 配置外部化（环境变量）
